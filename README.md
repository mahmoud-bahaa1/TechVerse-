About Techverse :-

Techverse is a simple educational platform created as part of an assignment for the Digital Egypt Pioneers Initiative (DEPI – رواد مصر الرقمية).

The main goal of this project is to practice and demonstrate front-end layout and responsiveness, with a strong focus on Bootstrap.

Techverse is a conceptual educational platform that aims to provide:

Learning resources

Tech-related content

A modern and clean UI for learners


The platform idea was chosen to apply Bootstrap components in a realistic educational context.
